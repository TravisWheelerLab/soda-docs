:trst-interface:`ChartRenderParams`
===================================

.. rst-class:: trst-interface-desc

A simple interface that defines the common parameters that should be used to render glyphs in any chart.

Properties
----------

----

:trst-property:`queryEnd`
+++++++++++++++++++++++++

.. rst-class:: trst-property-desc

The end coordinate of the region that will be rendered. 
 
.. rst-class:: trst-property-dl

:Type:
 .. rst-class:: trst-type

  number

:trst-property:`queryStart`
+++++++++++++++++++++++++++

.. rst-class:: trst-property-desc

:Type:
 .. rst-class:: trst-type
